# PRD

## 프로젝트명

**경영지원 운영 허브 (Management Support Ops Hub)**\
손샘학원 경영지원 통합 운영 플랫폼

------------------------------------------------------------------------

# 1. 프로젝트 목적

학원 운영 중 발생하는 다양한 경영지원 및 행정 업무를 단일 웹 플랫폼에서 통합 처리합니다.

주요 처리 업무: 
- **전산장비**: 구매 요청, 사양 최적화(조립 PC 빌더), 다단계 승인 프로세스
- **A/S 및 기술지원**: AI 기반 장애 자가 진단, FAQ 연동, 업체 티켓 자동 생성
- **소모품 관리**: 주변기기 장바구니 시스템, 다나와 실시간 최저가 연동
- **태블릿 렌탈**: 36개월 고정 약정 기반 전사 렌탈 자산 관리 및 연장/반납 프로세스
- **NAS 자산 관리**: 용량 모니터링, 사용자 권한 제어, 접속 가이드 자동화

핵심 목표:
표준화된 요청 → 스마트한 승인 → 실행 가이드 제공 → 데이터 기반 이력 저장

------------------------------------------------------------------------

# 2. 기술 스택

## Frontend
- Next.js 15 (App Router)
- TypeScript
- TailwindCSS
- Lucide Icons / shadcn/ui

## Backend
- Supabase (PostgreSQL, Auth, Realtime)
- Row Level Security (RLS) 적용
- Edge Functions (WebDAV/API Integration)

------------------------------------------------------------------------

# 3. 사용자 역할

1. **일반 사용자**: 지점 강사 및 행정 직원 (요청 접수, 진행 현황 조회)
2. **학원 관리자**: 지점 원장/실장 (1차 승인, 소액 결제 집행)
3. **경영지원팀**: 본사 운영 담당 (기술 검토, 업체 발주, 자산 등록)
4. **최고 관리자**: 회계팀 및 경영진 (고액 결제 최종 승인, 전사 지표 모니터링)
5. **NAS 관리자**: 서버 및 인프라 담당 (권한 설정, 용량 관리)

------------------------------------------------------------------------

# 4. 핵심 모듈

## MODULE A - 장비 구매 견적
조립 PC 본체 및 주요 전산 장비의 사양을 선택하고 승인 경로를 자동 라우팅합니다.
(요청 → 1차 승인 → 기술 검토 → 최종 승인 → 발주)

## MODULE B - A/S 자동화
사용자가 증상을 입력하면 AI가 해결책을 제시하며, 해결 불가능 시 자동으로 A/S 티켓을 생성합니다.
(증상 입력 → AI 진단 → FAQ 가이드 → 티켓 생성)

## MODULE C - 소모품·주변기기 (장바구니)
마우스, 키보드 등 소모품을 장바구니에 담아 통합 견적을 산출하고 장비 구매로 연동합니다.
(검색 → 장바구니 선택 → 장비 견적 합산 → 통합 승인)

## MODULE D - 태블릿 렌탈
36개월 고정 약정 기반의 렌탈 요청 및 모델 관리를 처리합니다.
(모델 선택 → 수량 입력 → 약정 확인 → 렌탈 실행)

## MODULE E - NAS 관리
실시간 NAS 용량 및 WebDAV 연결 상태를 모니터링하고 사용자 가이드를 제공합니다.
(현황 조회 / 권한 변경 / 접속 안내 자동화)

------------------------------------------------------------------------

# 5. 보안 및 감사
- **RBAC**: 역할 기반 접근 제어
- **RLS**: 테넌트(지점)별 데이터 격리
- **Audit Log**: 모든 승인 및 변경 이력 기록
- **AI Diagnosis**: 비정상적인 고액 요청에 대한 자동 감지 및 리포트
