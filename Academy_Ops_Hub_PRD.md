# PRD

## 프로젝트명

**Academy Ops Hub (AOH)**\
학원 경영지원 통합 운영 플랫폼

------------------------------------------------------------------------

# 1. 프로젝트 목적

학원 운영 중 발생하는 경영지원 업무를 단일 웹 플랫폼에서 처리합니다.

현재 수작업 업무: - 전산장비 구매 요청 - 장비 승인 - 업체 연결 - A/S
접수 - 자가 해결 가이드 - 태블릿 렌탈 신청 - 견적 요청 - 발주 승인 -
택배 요청 - NAS 사용자 관리 - NAS 접속 안내 - 장비 이력 관리

핵심 목표:

요청 표준화 → 승인 → 실행 → 이력저장 → 자동화

------------------------------------------------------------------------

# 4. 핵심 모듈

## MODULE A - 전산장비 구매 시스템

요청 → 승인 → 검토 → 최종승인 → 구매 → 완료

## MODULE B - 전산 A/S 자동화

증상 검색 → FAQ → 해결 → 실패 시 A/S 티켓 생성

## MODULE C - 부품 구매

검색 → 선택 → 승인 → 구매

## MODULE D - 태블릿 렌탈 자동화

요청 → 견적 → 승인 → 발주 → 배송 → 완료

## MODULE E - NAS 관리

용량 모니터링 / 사용자 추가 / 권한 변경 / RaiDrive 연결
