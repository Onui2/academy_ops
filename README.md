# 경영지원 운영 허브 (Management Support Ops Hub)

학원의 경영지원 업무를 통합 관리하는 스마트 운영 시스템입니다. 장비 구매, A/S 진단, 소모품 관리 및 태블릿 렌탈 등 모든 운영 프로세스를 디지털화하여 효율적인 경영 지원을 실현합니다.

## 핵심 모듈
- **장비 구매 견적**: 조립 PC 빌더 및 최적의 장비 사양 추천
- **A/S 자동 진단**: AI 기반 장애 진단 및 티켓 자동 생성
- **소모품·주변기기**: 실시간 다나와 연동 및 장바구니 기반 견적
- **태블릿 렌탈**: 36개월 고정 약정 기반의 전사 태블릿 관리
- **NAS 모니터링**: 실시간 저장소 상태 확인 및 권한 관리

## 기술 스택
- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: Supabase (Auth, Database, Storage)
- **Icons**: Lucide React
- **Deployment**: Vercel

## 시작하기
```bash
npm install
npm run dev
```
