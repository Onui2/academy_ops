# Academy Ops Hub - AI Harness PRD

## 개요

Academy Ops Hub 개발 및 운영을 위한 AI Harness 설계 문서입니다.

목표: - 개발 자동화 - 코드 검수 자동화 - 문서 자동 생성 - 장애 분석
자동화 - 운영 질의응답 자동화

------------------------------------------------------------------------

## 구조

사용자 요청 → Router AI → 역할별 AI 분배 → 결과 취합 → 검수 → 최종 출력

------------------------------------------------------------------------

## AI 역할

### 1. Router

요청 분석 후 적절한 모델로 분배

분류: - 코드 생성 - 디버깅 - 문서 작성 - DB 설계 - 보안 검토 - UI 설계

------------------------------------------------------------------------

### 2. Builder

실제 코드 생성

담당: - Frontend - Backend - API - SQL - Infra

------------------------------------------------------------------------

### 3. Reviewer

코드 검토

검토: - 버그 - 성능 - 보안 - 유지보수성

------------------------------------------------------------------------

### 4. Auditor

외부 감사 역할

검증: - 논리 오류 - 권한 누락 - 보안 위험 - 데이터 정합성

------------------------------------------------------------------------

### 5. Doc Writer

자동 문서화

생성: - README - API 문서 - 변경 이력 - 운영 매뉴얼

------------------------------------------------------------------------

## 개발 플로우

기획 → Router → Builder → Reviewer → Auditor → 승인 → Bitbucket PR →
Merge
